# voice-automation-assistant
## Overiew

A Python-based voice assistant that executes system and web commands using speech recognition and text-to-speech.

The assistant can retrieve information, automate desktop actions, and interact with online services.

## Features

* Voice input using SpeechRecognition
* Text-to-speech responses
* Wikipedia search and summaries
* YouTube playback
* Web search automation
* Open desktop applications (Notepad, Chrome)
* Real-time time reporting

## Tech Stack

* Python
* pyttsx3 (offline TTS)
* SpeechRecognition
* Wikipedia API
* PyWhatKit

## How to Run

1. Install dependencies:

   ```
   pip install -r requirements.txt
   ```

2. Run:

   ```
   python jarvis.py
   ```

3. Speak commands like:

   * "time"
   * "search Python"
   * "play music"
   * "open chrome"

## Purpose

This project explores voice interfaces, command parsing, and desktop automation using Python.
